# Handtalk Accessibility Toolkit
Toolkit desenvolvido com React e Typescript
## Setup

- [Node.js](https://nodejs.org/pt-br/)
- [Npm](https://nodejs.org/pt-br/)

## Instalação

No diretorio da aplicação use:

### `npm install`

E para iniciar a aplicação:

### `npm start`
